# Format a JSONC file

Format a JSONC file

## Usage

``` r
ts_format_jsonc(file = NULL, text = NULL, options = NULL)
```

## Arguments

- file:

  Path to a JSONC file. Use one of `file` or `text`.

- text:

  JSONC content as a raw vector or a character vector. Use one of `file`
  or `text`.

- options:

  Named list of parsing and formatting options, see [tsjsonc
  options](https://gaborcsardi.github.io/tsjsonc/reference/tsjsonc_options.md).
